Description of Available Resources and URIs

  1. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant

    • This get method endpoint retrieves all of the vancouver restaurants in the api with their details (name, address, postal code, status, cuisine)

  2. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant

    • This post method endpoint creates a new Vancouver restaurant

  3. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant/:id

    • This get method endpoint returns the Vancouver restaurant specified by id

  4. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant/:id

    • This put method endpoint updates a Vancouver restaurant specifed by id

  5. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant/:id

    • This delete method endpoint deletes a Vancouver restaurant specifed by id

  6. https://backend-restaurant-termproject.herokuapp.com/vancouverRestaurant

    • This delete method endpoint deletes all Vancouver restaurants

  7. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant

    • This get method endpoint retrieves all of the Burnaby restaurants in the api with their details (name, address, postal code, status, cuisine)

  8. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant

    • This post method endpoint creates a new Burnaby restaurant

  9. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant/:id

    • This get method endpoint retrieves one Burnaby restaurant specifed by id

  10. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant/:id

    • This put method endpoint updates one Burnaby restaurant specifed by id

  11. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant/:id

    • This delete method endpoint deletes one Burnaby restaurant specifed by id

  12. https://backend-restaurant-termproject.herokuapp.com/burnabyRestaurant

    • This delete method endpoint deletes all Burnaby restaurants

  13. https://backend-restaurant-termproject.herokuapp.com/user

    • This post method endpoint inserts a new user with their email, promo code, and token

  14. https://backend-restaurant-termproject.herokuapp.com/users

    • This get method endpoint retrieves all of the current users

  15. https://backend-restaurant-termproject.herokuapp.com/user/:id

    • This get method endpoint retrieves a specific user by id

  16. https://backend-restaurant-termproject.herokuapp.com/user/:id

    • This put method endpoint updates a specific user by id

  17. https://backend-restaurant-termproject.herokuapp.com/user/:id

    • This delete method endpoint deletes a specific user by id

  18. https://backend-restaurant-termproject.herokuapp.com/users

    • This delete method endpoint delets all users

Available HTTP methods

  • POST
  • GET
  • PUT
  • DELETE
